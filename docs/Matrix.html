Class: Matrix

Matrix(values)

new Matrix(values)

2D 3x2 transformation matrix, used to represent linear geometric transformations over objects. The values of the matrix are stored as numeric array. The matrix can be applied to the canvas or DOM elements using CSS transforms.
Parameters:
Name Type Description
values Array.<number> Array of matrix values by row, needs to have exactly 6 values.
